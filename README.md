# Ectra Service Center

تطبيق React (Vite) لمركز خدمة منتجات ECTRA by Kieslect — بإدارة MTC GROUP.
متصل بـ Supabase (مشروع Agency backend).

## تشغيل محلي
```
npm install
npm run dev
```

## نشر على Vercel
ارفع المجلد على GitHub ثم Import المشروع في Vercel (هيتعرّف على Vite تلقائياً).

## نشر على Netlify
Build command: `npm run build` — Publish directory: `dist`

## إصدار 4014
- صفحة العميل: قسم «آراء العملاء» تحت فورم التقييم — يعرض التقييمات المنشورة فقط، 10 في الصفحة مع تنقّل، ويتحدّث بعد إرسال تقييم جديد.
- الأدمن (التقييمات): زر إظهار/إخفاء لكل تقييم — المخفي يظهر باهت وبعلامة «مخفي عن العملاء». (يستخدم صلاحية feedback/delete نفسها — مفيش صلاحية جديدة، فمش محتاج Logout/Login).
- ختم الضمان: استبدال الصورة بنسخة نظيفة بدون علامة مائية (`src/assets/warranty-stamp.png`).
- الدوال المطلوبة على Supabase موثّقة في `sql/feedback_4014_contract.sql` (مطبّقة Live بالفعل).

## تعديل / حذف شهادات الضمان
- في «قائمة الضمانات» بقى فيه زر تعديل (✎) وحذف (🗑) لكل شهادة — يظهروا حسب الصلاحية.
- اتضافت صلاحيتين جديدتين تحت «قائمة الضمانات»: «تعديل الشهادة» و«حذف الشهادة» — المدير يمنحهم للمستخدمين من شاشة إدارة الصلاحيات.
- **مطلوب مرة واحدة:** شغّل `sql/warranty_manage.sql` على Supabase (بينشئ دالتي ectra_warranty_update / ectra_warranty_delete ويمنح الصلاحية للأدوار الإدارية)، وبعدها Logout/Login.

## منع تكرار شهادة الضمان لنفس المنتج
- لا يمكن إصدار أكثر من شهادة ضمان لنفس السيريال. لو حاولت، بتظهر رسالة: «هذا المنتج له شهادة ضمان بالفعل (رقمها …)».
- الحارس مُطبّق في `sql/warranty_manage.sql` (القسم 4) كـ trigger على جدول الضمانات — يشتغل بعد تشغيل الملف.
- التكرارات الموجودة مسبقاً تظل كما هي؛ في نهاية ملف الـ SQL استعلام (معطّل بـ --) يعرض السيريالات المكررة لتنظيفها يدوياً.

## ترقيم قائمة الضمانات (حقيقي من القاعدة)
- القائمة بقت بترقيم حقيقي: 20 ضمان في الصفحة (قابل للتعديل من ثابت WARR_PER_PAGE)، مع أزرار السابق/التالي وإجمالي العدد.
- البحث بقى على مستوى القاعدة (يدوّر في كل الضمانات مش الصفحة الحالية بس) بتأخير بسيط.
- **مطلوب:** شغّل القسم (5) من `sql/warranty_manage.sql` (دالة ectra_warranty_page). لو شغّلت الملف كامل تاني فهو آمن (create or replace).
